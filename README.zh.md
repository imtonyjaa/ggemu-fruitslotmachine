## 🔥 技术特性

作为 GGEMU 旗舰级 H5 游戏，本项目展示了多项核心技术：

*   **🎰 经典 9 线连线机制**：
    还原真实的复古水果机逻辑，支持多重下注与精准的停止判定。

*   **⚡️ 丝滑滚轮动画**：
    高度优化的滚动运动学算法，利用定制缓动函数实现逼真的物理惯性与机械回弹效果。

*   **🧮 专业 RTP 概率引擎**：
    企业级的返奖（RTP）算法，完美平衡玩家的中奖刺激感与平台的经济系统。

*   **🔊 复古街机音效**：
    多声道的经典 8-bit 及赌场音效，营造极具沉浸感的怀旧体验。

### 2. 📸 高清媒体捕获
