## 🔥 Technical Features

As a flagship HTML5 game for GGEMU, this project showcases several robust technologies:

*   **🎰 Classic 9-Line Slot Mechanics**:
    Authentic retro fruit slot logic with precise reel stops and multi-bet winning combinations.

*   **⚡️ Smooth Reel Animations**:
    Highly optimized spin kinematics utilizing custom easing functions for realistic momentum and stopping bounce.

*   **🧮 Tuned RTP Probability Engine**:
    A professional-grade payout algorithm that balances player excitement with platform economy using Return to Player (RTP) design.

*   **🔊 Retro Arcade Audio**:
    Multi-channel classic chiptune and casino sound effects ensuring an immersive nostalgic experience.
